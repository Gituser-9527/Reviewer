# 实施任务清单

## 使用说明

- 状态标记：`[ ]` 未开始、`[x]` 完成
- 每项任务应关联负责人、验收证据和必要的设计/法规审批
- 法规内容、阈值和规则发布必须由有权限的法务/合规人员确认
- 阶段按依赖顺序排列；可并行项仍需遵守契约先行

## Phase 0：决策与准备

- [ ] 确认首发司法辖区、平台范围、语言和目标客户
- [ ] 指定法务负责人、规则审批人、人工复核负责人和数据负责人
- [ ] 决定数据保留、加密、脱敏、模型零保留和数据驻留要求
- [ ] 确认 MVP 是否仅单条同步审核，以及人工复核 SLA
- [ ] 建立 ADR 目录并确认关键架构决定
- [ ] 定义 MVP 风险分类、正式依据清单和首批黄金样本

完成标准：所有 PRD 待决策项有负责人和结论，首批规则可进入编写。

## 暂不实施

- 自动抓取后未经审批直接发布法规规则
- 由 LLM 自主创建并启用拦截规则
- 将模型生成条款作为正式法律依据
- 使用生产岗位或候选人数据训练模型而没有明确授权
- 自动发布、下架或处罚企业而没有外部系统的明确授权与复核策略
