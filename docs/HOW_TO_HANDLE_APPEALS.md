# How To Handle Appeals

## 处理原则

申诉处理必须由人工复审员做最终决定。Appeal Agent 只能辅助整理两侧理由，不得自动推翻原结论。

## 步骤

1. 查看原审核结论、ruleId、evidenceId 和 matchedText。
2. 阅读企业申诉理由和补充说明。
3. 检查是否已修改岗位文案。
4. 核对依据是否适用当前风险类别和地区。
5. 输出复审结论：维持、撤销或要求修改。
6. 对误杀、依据错误或规则过宽样本，创建规则改进建议。

## 可接受表述

- 原结论依据不足，建议调整为人工复核。
- 企业已修改风险表达，建议重新审核修改后文案。
- 当前证据仍支持要求修改。

## 禁止表述

- 企业已经违法。
- 申诉必然成立。
- 系统自动撤销原结论。
