# 评测计划

## 1. 评测目标

评测系统是否能准确识别风险、做出安全决策、提供真实依据和可用改写，并在模型或依赖异常时保持保守行为。高风险漏报的成本高于普通误报，但人工复核量也必须可运营。

## 2. 评测对象

- 规则引擎单规则与聚合决策
- LLM 语义风险识别
- 最终决策器
- 依据引用
- 修改建议与合规改写
- 人工复核流程
- 性能、稳定性、安全和成本
- 后续 RAG 检索

## 3. 数据集设计

### 数据来源

- 法务/运营编写的规则正反例
- 经授权并去标识化的真实岗位样本
- 历史误报、漏报和人工推翻样本
- 对抗样本：谐音、拆字、空格、图片转文本、委婉表达和上下文否定
- 合成样本：仅用于补充覆盖，必须标记来源且不能替代真实分布

### 数据切分

- `development`：规则和提示开发
- `validation`：阈值选择与版本门禁
- `test`：冻结盲测集，仅发布评测使用
- `challenge`：持续扩充的高难与对抗集

同一模板、企业或近重复文本不得跨集合泄漏。

### 标注结构

每条样本至少包含：

- 适用地区、平台、审核时间和输入字段
- 期望结论与风险等级
- 风险类别、规则 ID、证据范围
- 适用依据及版本
- 是否允许自动改写、期望修改原则
- 标注人、复核人、分歧处理记录

高风险样本建议双人独立标注，分歧由法务仲裁。

## 4. 核心指标

### 决策指标

- 三分类混淆矩阵：`PASS`/`BLOCK`/`REVIEW`
- 高风险漏放率：期望 `BLOCK` 或 `REVIEW` 却输出 `PASS`
- 自动拦截准确率：输出 `BLOCK` 的样本中真正应拦截比例
- 人工复核率与人工推翻率
- 风险等级准确率和平均偏差

### Finding 指标

- 按类别的 precision、recall、F1
- 证据定位准确率/重叠度
- 规则 ID 与依据匹配准确率
- 重复 finding 比率

### 改写指标

- 风险消除率：改写后二次检查不再命中目标风险
- 事实保持率：不编造或删除岗位核心事实
- 信息完整性与可读性
- 不安全改写率：仍违规却标记为已合规

改写质量需要规则自动检查加人工量表，不能只用 LLM 自评。

### 系统指标

- P50/P95/P99 延迟、超时率和错误率
- LLM token、成本和重试率
- 同输入重复运行一致性
- Provider 故障时安全降级成功率

## 5. MVP 发布门槛

具体数值需结合首批数据和法务风险偏好确认，初始建议：

- `CRITICAL` 类召回率 >= 99%，且冻结测试集不得出现可复现的自动放行
- `HIGH` 类召回率 >= 95%
- 所有 `BLOCK`/`REVIEW` 的证据可定位率 >= 98%
- 正式依据引用准确率 = 100%；无法确认时不得生成依据
- 不安全改写率 <= 1%，关键事实编造率 = 0%
- LLM 完全不可用时，需语义判断的样本 100% 不自动放行
- P95 满足 PRD 的 5 秒目标，或明确启用异步模式

这些是工程门禁建议，不代表法律风险已被完全消除。

## 6. 测试层次

1. YAML schema 和静态检查
2. 单规则正反例测试
3. 规则组合、优先级和冲突测试
4. LLM Provider 契约与结构化输出测试
5. 审核管线集成测试
6. API、权限、幂等和并发测试
7. 离线黄金集评测
8. 影子流量和人工对照评测
9. 灰度上线与持续监控

## 7. 模型与提示版本评测

任何模型、系统提示、温度、结构化 schema 或路由策略变化，都视为待评测版本。报告必须记录：

- 代码 commit、规则版本、模型和提示版本
- 数据集版本与样本数
- 全局指标及按类别/地区/平台切片
- 相对基线的提升与退化
- 新增误报和漏报案例清单
- 成本、延迟和稳定性变化

未经门禁批准不得直接替换生产版本。

## 8. 鲁棒性与安全评测

- Unicode、全半角、零宽字符和混合语言
- 否定语境、引用违规文本和教育性文本
- Prompt injection 与要求模型忽略规则
- 超长文本、重复文本和恶意正则输入
- 模糊薪资、联系方式、导流和变体词
- 跨租户数据泄漏和越权查询
- 日志、错误响应与模型请求中的敏感信息泄漏

## 9. 线上反馈闭环

- 人工推翻自动进入待分析队列，不直接成为训练真值
- 每周分析高频规则、误报、漏报和复核积压
- 经法务确认后纳入新数据集版本
- 监控结论分布漂移、规则命中突变和模型成本异常
- 保留生产版本回滚能力

## 10. 评测产物

- 版本化数据集 manifest
- 机器可读指标 JSON
- 人可读评测报告
- 失败案例与回归用例
- 发布门禁结论及审批记录
