# Common Misjudgment Cases

## 1. 把岗位必要条件误判为歧视

示例：夜班安保岗位要求能适应夜间巡逻。  
处理：不要直接标记为歧视，检查是否出现性别、婚育、年龄等与履职无关限制。

## 2. 把企业福利误判为收费

示例：公司提供工服。  
处理：只有出现“缴纳、押金、扣费、贷款”等劳动者负担时，才考虑收费风险。

## 3. 把联系方式误判为隐私过度收集

示例：简历投递邮箱。  
处理：普通投递邮箱通常不是过度收集；身份证号、银行卡号、家庭住址等更敏感。

## 4. 把高薪表达全部判为虚假

示例：综合薪资 15k-25k。  
处理：需要结合是否有“轻松月入十万、保底高薪无需经验”等夸大表达。

## 5. 改写文案仍保留风险词

示例：原文“限女性”，改写后仍有“女性优先”。  
处理：标记 `BAD_REWRITE`，不要加入通过样本。
