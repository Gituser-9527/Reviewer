# Operator FAQ

## 如何判断系统能否进入 assist 模式？

Go / No-Go 必选项应全部 `pass` 或经过合规负责人明确 `waived`，且最近日报无 P0/P1 阻塞。

## 什么时候不要把样本加入评估集？

- 人工标签未统一。
- 反馈描述不清楚。
- 样本含未脱敏个人敏感信息。
- 争议样本尚未由高级审核员裁决。

## 反馈类型选错怎么办？

不要直接删除历史反馈。追加新反馈或通过人工复核闭环修正，并在日报中记录。

## limited_enforce 可以给所有租户开吗？

不可以。只能对明确授权的 tenant 和低争议场景开启，且必须保留回滚路径。
