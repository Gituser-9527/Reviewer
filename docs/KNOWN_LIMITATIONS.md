# Known Limitations

## 当前限制

- Beta 阶段不提供法律裁判结论，只输出合规风险提示和审核建议。
- PDF 报告仍是 MVP 占位导出，正式版需要版式渲染和验收。
- RAG 知识库依赖本地维护内容，法规更新需要人工确认。
- 隐晦表达和对抗样本仍需依赖人工复核闭环持续提升。
- `limited_enforce` 仅允许对指定 tenant 和低争议规则开启。
- 客户反馈和日报当前为应用内 MVP 存储，生产环境需接 PostgreSQL repository。

## 禁用场景

- 不得用于处罚企业的唯一依据。
- 不得自动发布法规知识库或规则。
- 不得把模型生成内容当作权威法规依据。
- 不得在日志、反馈或测试样本中保存敏感信息明文。
