# Beta Launch Plan

## 目标

让内部审核员、运营人员和合规人员在受控范围内使用招聘合规审核 Agent，验证审核准确性、解释质量、人工复核流程和运营价值。

## 范围

- 首批仅面向指定 tenant 和指定使用人员。
- 支持 `shadow`、`assist`、`limited_enforce` 三种模式。
- Beta 结论用于辅助审核，不作为法律裁判结论。

## 阶段

1. 准备期：确认参与人员、SOP、已知限制、升级通道和 Go/No-Go 检查。
2. Shadow：Agent 只记录结果，不影响线上业务。
3. Assist：Agent 结果展示给人工审核员，人工做最终决定。
4. Limited Enforce：仅对指定 tenant、指定低争议场景开启有限自动处置。
5. 复盘：汇总日报、反馈、误杀漏判、ROI 和是否扩大试点。

## 退出条件

- 出现无法解释的高风险误判。
- 发现敏感信息明文泄露。
- Go/No-Go 必选项失败。
- 反馈中出现未闭环的重大流程缺口。
